=head1 DEVELOPER NOTES

Please do not use locale translations when creating new iContact module schemas.
Currently, this can cause upcp updates to break, as the lh->maketext() call will
be accessed from outside of the upcp bootstrapping process.

Ref. CPANEL-40890
