=head1 NAME

install_best_available_certificate_for_domain

=head1 SYNOPSIS

    install_best_available_certificate_for_domain ( <domain> | --help )

This command will install the best available certificate for a given
domain.

We define the best available certificate as the one that matches the DNS
name, is signed by a recognized CA, and has the longest validity period.
